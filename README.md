# Metadata Extractor (EXIF)

## 📖 Description
Metadata Extractor is a simple Python project that extracts EXIF metadata from image files. It helps users view hidden information such as camera model, image resolution, date and time, and other available metadata.

## ✨ Features
- Extract EXIF metadata from images
- Display camera information
- Show image details
- Easy to use
- Built with Python

## 🛠️ Technologies Used
- Python
- Pillow (PIL)

## ▶️ How to Run
1. Install Pillow:
   ```
   pip install Pillow
   ```
2. Run the program:
   ```
   python main.py
   ```
3. Enter the image path when prompted.
